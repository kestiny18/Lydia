# Built-in Skills

This folder contains built-in skills shipped with Lydia.

## Guidelines
- Use English only.
- Include YAML frontmatter.
- Keep steps concise and safe.

## Built-in Skills
- `hello-world`: basic wiring check for skill loading.
- `system-health-check`: quick diagnostic workflow.
- `git-advanced-workflow`: common advanced git process guidance.
- `node-project-management`: Node.js setup/install/run workflow guidance.
- `docker-container-management`: common container lifecycle workflow guidance.
- `webapp-testing`: Playwright-based web testing workflow.
- `browser-automation-workflow`: browser computer-use workflow (recon -> action -> verify).
