# Getting Started

This guide helps you initialize Lydia on a new machine.

## Quick Init
```bash
lydia init
```

This creates:
- `~/.lydia/config.json`
- `~/.lydia/strategies/default.yml`
- `~/.lydia/skills/`

## Run a Task
```bash
lydia run "check git status"
```

## Launch Dashboard
```bash
lydia dashboard
```
